eufy Security AddOn für HomeMatic

Dieses AddOn ermöglicht die Steuerung Ihres eufy Security Systems mit Ihre CCU.

Sie finden auf dieser Webseite alle Basisstationen und Kameras, die für den in den Einstellungen hinterlegten eufy Security Account verfügbar sind. Es werden Informationen zu den Geräten angezeigt und Sie können den aktuellen Modus Ihrers eufy Security Systems überprüfen und ändern.

Mit Hilfe der durch das AddOn bereitgestellten API haben Sie die Möglichkeit, den Status Ihrer Geräte auch in Ihrer CCU abbilden zu können und Sie können ebenfalls den Modus überprüfen und ändern. Dazu gibt es verschiedene Endpunkte, die in der Hilfe beschrieben werden.
